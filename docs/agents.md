# Agents 说明

本目录用于集中存放与自定义 agent 相关的文档与使用说明（集中放置以便团队查阅）。

## 已注册的 Agent

- **工作台助理** — `.github/agents/workspace-assistant.agent.md`
  - 作用：在与项目相关的工作中沉淀规则、生成模板、协助编写和修改代码。
  - 触发关键词：工作台助理、规则、skills、写代码、项目相关

## 快速开始

1. 在聊天中选择或输入 `工作台助理` 以唤起该 agent。
2. 示例提示：
   - "工作台助理：为此仓库生成代码审查规则，重点关注安全和可维护性"
   - "工作台助理：帮我为 `src/` 添加单元测试骨架，使用 pytest"

## 存放约定

- agent 文件：放在 `.github/agents/`
- 快速提示（prompts）：放在 `.github/prompts/`
- 团队文档：放在 `docs/` 下，便于静态站点或仓库浏览器展示
